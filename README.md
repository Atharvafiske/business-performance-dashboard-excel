# business-performance-dashboard-excel
This Excel dashboard visualizes key business metrics including:
- Total profit by payment method  
- Revenue by product line  
- Average expenses by department  
- Regional transaction counts  

📊 Built using pivot tables and charts for quick insights into operational and financial performance.  
💡 Ideal for showcasing data analytics and visualization skills using Excel.

## Files
- `Business_Performance_Dashboard.xlsx` — main dashboard file  
- `Data.xlsx` — raw transaction dataset  

## How to Use
1. Open the Excel file.
2. Navigate to the **Dashboard** sheet.
3. Interact with slicers and filters to explore insights.
